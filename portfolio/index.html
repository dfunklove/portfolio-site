Sparks React

Made for music therapists to track their lessons and record student progress. It began as a Ruby on Rails app for my client Sparks for Success; now freshly rebuilt in React for this portfolio. The server side is a GraphQL API running in Node.js using Apollo, Pothos, and Prisma.

Source code: React and GraphQL

Complete Discography

If you follow an artist who performs in several different bands, this tool is for you. Enter their name to get a list of every album they've released under all of their various bands and aliases.

This app uses Python and Flask to retrieve data from Discogs.com. The client and server communicate using Socket.IO, a full-featured websocket library.
